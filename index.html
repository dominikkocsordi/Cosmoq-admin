Admin-Rollen verwalten
Wie werden Admin-Rollen vergeben?
Admin-Rollen werden über app_metadata.role = "admin" in Supabase Auth gesetzt. Dies ist nur per Service Role Key oder direkt im Supabase Dashboard möglich — normale Nutzer können ihre eigene Rolle nicht ändern.

Wege: Supabase Dashboard → Authentication → Users → User auswählen → Edit → app_metadata: {"role":"admin"}
